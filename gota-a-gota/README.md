# App Gota a Gota

Sistema de microcréditos y préstamos diarios.

## Características

- ✅ Gestión de clientes
- ✅ Creación y seguimiento de préstamos
- ✅ Pagos diarios/semanales
- ✅ Reportes financieros
- ✅ Dashboard administrativo
- ✅ Notificaciones automáticas
- ✅ Historial de transacciones

## Stack Tecnológico

- **Backend**: Node.js + Express + MongoDB
- **Frontend**: HTML5 + CSS3 + JavaScript (Vanilla)
- **Autenticación**: JWT
- **Base de datos**: MongoDB Atlas

## Estructura del Proyecto

```
gota-a-gota/
├── backend/
│   ├── models/
│   ├── routes/

│   └── server.js
├── frontend/

│   └── index.html
└── docs/
```

## Instalación

```bash
cd gota-a-gota/backend
npm install
npm run dev
```
